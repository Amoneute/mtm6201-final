# Apex Lens Photography Website

## Overview
Apex Lens is a responsive photography portfolio website built using HTML, CSS, and Bootstrap 5. The website includes a homepage, portfolio gallery, and contact page. It is designed to showcase photography work in a clean, modern, and mobile-friendly layout.

---

## 🎯 Process (How I Built the Website)

I started by planning the structure of the website using a simple layout: a homepage, a portfolio page, and a contact page. I then used HTML to build the structure and Bootstrap 5 to create a responsive grid system.

After that, I added custom CSS to improve the visual design, including spacing, image styling, and hover effects. I focused on making sure the layout was consistent across all pages and that images displayed properly on different screen sizes.

I also organized my images into a folder structure and made sure all file paths were correct so everything would load properly on GitHub Pages.

---

## ⚠️ Challenges Faced

### 1. Image sizing inconsistency
Some images appeared different in height and broke the alignment of the gallery.

**Solution:**
I solved this by using `aspect-ratio` and `object-fit: cover` in CSS to ensure all images maintained the same size and cropped consistently.

---

### 2. Broken image paths on GitHub Pages
Some images did not display correctly when the site was deployed.

**Solution:**
I fixed this by checking folder names and correcting case sensitivity issues, since GitHub Pages is case-sensitive.

---

### 3. Responsive layout issues
At first, the layout did not look good on mobile devices.

**Solution:**
I used Bootstrap grid classes like `col-12`, `col-md-4`, and tested the layout in different screen sizes to ensure responsiveness.
